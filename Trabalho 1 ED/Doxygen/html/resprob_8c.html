◆ eval_exp()

double eval_exp ( char *  expr)

Faz a avaliação e resolução da string posfixa.

Recebe a string da função anterior, e inicia uma pilha de números reais; Caso a parte da string até o próximo ' '(espaço) seja um operando, este é então empilhado já como número real. Caso essa parte seja um operador, dois valores da pilha são desempilhados e então ocorre a operação correspondente, empilhando o resultado na pilha. Isso ocorre até que a string posfixa acabe.

◆ inf_to_postf()

void inf_to_postf ( char *  exp_infix,
char *  exp_postfix 
)

Converte a string infixa para posfixa.

Recebe a string infixa e a string vazia posfixa da função anterior, e inicia uma pilha de caracteres. Percorre a string infixa, e caso o caractere ou parte da string seja um operando, copia o mesmo para a string posfixa. Caso seja um operador, o empiha. Caso tenha prioridade maior que algum já empilhado, desempilha o de menos prioridade e copia para a posfixa. Caso encontra um parênteses de abertura, o empilha. Caso encontre um de fechamento, desempilha e copia na saída até que seja desempilhado o símbolo de abertura.

double strtolf ( char *  expr)

Transforma uma string em um número real.

Recebe a string da função anterior, varre a string à procura de um '.' ou ','; os quais determinam que um número possui parte inteira e parte fracionária. Então, a partir deste ponto, caso exista, é começada a conversão, char por char, trabalhando com potências de 10 positivas e negativas.
